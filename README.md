# GameCursor

>image folder

place all your png cursor, if you don't have one the program will create one from the config file.

>config file

* midle: none
* color: (all text color: red)
* size: (number)px
* diviseur : (number) 
> Instalation 

Unzip the GameCursor file, start setup.exe and the software will be ready to use
